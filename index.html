검색 도움말

1. 기간검색

검색창에 원하는 시작일과 종료일을 입력하면, 해당 기간에 발행된 모든 자료를 조회할 수 있습니다.

  • 키워드를 입력하지 않은 경우 → 지정한 기간 내 전체 기사가 검색됩니다.
  • 키워드와 함께 입력한 경우 → 해당 기간 중 입력한 키워드를 포함한 기사만 표시됩니다.

2. 키워드검색

기간을 지정하지 않고 키워드만 입력하면, 모든 기간의 자료 중 키워드와 일치하는 결과를 보여줍니다.

  • 키워드에 “독립신문”을 입력하면 모든 연도의 ‘독립신문’ 관련 기사가 검색됩니다.

3. 검색 예시

  • 기간: 19190401 ~ 19191231 / 키워드: ‘상해’ → 1919년 상해 관련 기사만 표시
  • 기간: 19200101 ~ 19201231 / 키워드 비움 → 1920년 전체 기사 표시
  • 기간 미입력 / 키워드: ‘임시정부’ → 전체 연도 중 임시정부 관련 기사 표시

    주제 큐레이션

    도시는 바뀌어도 나라는 있다

    임정의 여정, 상하이에서 충칭까지

    1919년 4월, 대한민국임시정부는 중국 상하이에 첫 발을 내디뎠다. 망명이라는 이름의 독립운동은 그저 피신이나 은신이 아니었다.

    망명정부의 법과 질서

    임시헌장과 그 너머

    헌법 초안, 회의록, 포고문 속에서 국가는 존재했다. 망명 중에도 자치와 통치를 멈추지 않았던 기록을 찾아 떠난다

    독립은 공짜가 아니었다

    후원과 모금의 기록들

    외교만으로는 부족했다. 임시정부는 스스로 군을 조직했고, 그들의 총은 해방을 향해 조준되어 있었다

    총을 든 임정

    한국광복군의 탄생과 작전 보고

    의연금, 애국공채, 교포의 헌금까지. 모든 투쟁에는 자금이 있었고, 모든 자금엔 이름 없는 손이 있었다의연금, 애국공채, 교포의 헌금까지. 모든 투쟁에는 자금이 있었고, 모든 자금엔 이름 없는 손이 있었다
